# my-profile

[English](./docs/lang/en.md) | 日本語

個人プロフィールサイト

## 機能紹介

- **レスポンシブデザイン**: モバイル・デスクトップ対応
- **ダークモード**: システム設定に応じた自動切り替え
- **モダンなUI**: Tailwind CSS v4による最新スタイリング
- **型安全性**: TypeScriptによる完全な型チェック

## 導入手順

### リポジトリのクローン

```bash
git clone <repository>
cd my-profile
```

### 依存関係インストール

```bash
pnpm install
```

### サーバー起動

```bash
pnpm dev
```

ブラウザで `http://localhost:3000` を開いて確認してください。
